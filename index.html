Objetivos

General y especificos

Objetivo General

* Usar correctamente las etiquetas básicas para la creación de páginas web en lenguaje HTML y CSS3, estructurando correctamente su diseño a través de la maquetación usando un software apropiado.

Objetivos Especificos

* Emplear la estructuración de contenido usando las etiquetas de SECTION, ASIDE, FOOTER, ARTICLE.
* Implementar etiquetas para la organizacion del texto tales como strong, em ,a href
* Usar etiquetas de resolucion de caracteres, fuentes, colores, fondo
